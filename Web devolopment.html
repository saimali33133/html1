Web application

From Wikipedia, the free encyclopedia

A web application (or web app) is application Software that runs on a web server, unlike computer-based software programs that are run locally on the operating system (OS) of the devise.Web application are accessed by the user through a web browser with an active network connection.These applications are programmed using a client-server modeled structure-the user ("client") is provided services throgh an off-site server that is hosted by a third-party.Examples of commonly-used web applications include: web-mail,online retail sales,online banking, and online auctions.
